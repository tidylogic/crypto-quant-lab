# Research Notes

Use this directory for research logs that compare ideas across time.

Recommended file naming:

```text
YYYY-MM-DD-topic.md
```

Each note should include:

- Hypothesis.
- Strategy or FreqAI model touched.
- Timerange and pairlist.
- Feature/target changes.
- Backtest result link or file path.
- Decision: reject, revise, or promote to dry-run.
